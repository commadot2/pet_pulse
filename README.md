### Description
A regular one-page website that utilizes:
- CSS Selectors
- CSS Pseudo classes
- SCSS
- Animations (wow.js, animate.css, tns carousel)
- Fonts
- Responsive/Adaptive
- Gulp
- Bootstrap
- BEM principles
- SEO optimization
- Google maps integration
